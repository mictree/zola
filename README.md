Zola is a social media for Vietnamese.

## Demo


## Tech Stack

**Client:** React, React Query, TailwindCSS

**Server:** Node, Express, Socket IO

**Mobile:** Flutter

**Database:** MongoDB

## Running Tests

To run tests, run the following command

```bash
  yarn test
```

## Features

- Post text, image or video
- Comment to post
- Chat with friend
- Follow other user
- Receive notification
